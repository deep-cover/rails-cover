6 # == Multibyte proxy
7 #
8 # +mb_chars+ is a multibyte safe proxy for string methods.
9 #
10 # It creates and returns an instance of the ActiveSupport::Multibyte::Chars class which
11 # encapsulates the original string. A Unicode safe version of all the String methods are defined on this proxy
12 # class. If the proxy class doesn't respond to a certain method, it's forwarded to the encapsulated string.
13 #
14 # >> "ǉ".upcase
15 # => "ǉ"
16 # >> "ǉ".mb_chars.upcase.to_s
17 # => "Ǉ"
18 #
19 # NOTE: An above example is useful for pre Ruby 2.4. Ruby 2.4 supports Unicode case mappings.
20 #
21 # == Method chaining
22 #
23 # All the methods on the Chars proxy which normally return a string will return a Chars object. This allows
24 # method chaining on the result of any of these methods.
25 #
26 # name.mb_chars.reverse.length # => 12
27 #
28 # == Interoperability and configuration
29 #
30 # The Chars object tries to be as interchangeable with String objects as possible: sorting and comparing between
31 # String and Char work like expected. The bang! methods change the internal string representation in the Chars
32 # object. Interoperability problems can be resolved easily with a +to_s+ call.
33 #
34 # For more information about the methods defined on the Chars proxy see ActiveSupport::Multibyte::Chars. For
35 # information about how to change the default Multibyte behavior see ActiveSupport::Multibyte.
36 def mb_chars
37 ActiveSupport::Multibyte.proxy_class.new(self)
38 end

40 # Returns +true+ if string has utf_8 encoding.
41 #
42 # utf_8_str = "some string".encode "UTF-8"
43 # iso_str = "some string".encode "ISO-8859-1"
44 #
45 # utf_8_str.is_utf8? # => true
46 # iso_str.is_utf8? # => false
47 def is_utf8?
48 case encoding
49 when Encoding::UTF_8
50 valid_encoding?
51 when Encoding::ASCII_8BIT, Encoding::US_ASCII
52 dup.force_encoding(Encoding::UTF_8).valid_encoding?
53 else
54 false
55 end
56 end
