42 # Did this attribute change when we last saved? This method can be invoked
43 # as +saved_change_to_name?+ instead of <tt>saved_change_to_attribute?("name")</tt>.
44 # Behaves similarly to +attribute_changed?+. This method is useful in
45 # after callbacks to determine if the call to save changed a certain
46 # attribute.
47 #
48 # ==== Options
49 #
50 # +from+ When passed, this method will return false unless the original
51 # value is equal to the given option
52 #
53 # +to+ When passed, this method will return false unless the value was
54 # changed to the given value
55 def saved_change_to_attribute?(attr_name, **options)
56 mutations_before_last_save.changed?(attr_name, **options)
57 end

59 # Returns the change to an attribute during the last save. If the
60 # attribute was changed, the result will be an array containing the
61 # original value and the saved value.
62 #
63 # Behaves similarly to +attribute_change+. This method is useful in after
64 # callbacks, to see the change in an attribute that just occurred
65 #
66 # This method can be invoked as +saved_change_to_name+ in instead of
67 # <tt>saved_change_to_attribute("name")</tt>
68 def saved_change_to_attribute(attr_name)
69 mutations_before_last_save.change_to_attribute(attr_name)
70 end
